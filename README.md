# 412Park - Pitt Parking Finder

Real-time parking availability app for University of Pittsburgh commuters.

## Team Members
- **Scrum Master:** Shihab Al Zakwani
- **Product Owner:** Ali Ammad
- **Developers:** Yang Ye, Cole Goucher, Pingzhao Li

412Park is a parking application designed to help Pitt commuters quickly find available parking near campus. The system will display real-time parking information, let users report open or occupied spots, show parking fees, and provide navigation to available spaces.
